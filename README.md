## O sistema deve ser capaz de:

**Clientes** 
 - Adicionar
 - Editar
 - Remover
 - Ler

**Produtos**
 - Adicionar
 - Editar
 - Remover
 - Ler
 
**Vendas**

     - O usuário do sistema deve poder conceder descontos, em porcentagem ou em valores reais.
